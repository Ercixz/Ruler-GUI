## 概念模型

```
最终 AGENTS.md = 全局拼图块 ⊕ 项目拼图块 ⊕ 自定义块

全局拼图块（所有项目共享）          项目拼图块（仅当前项目）
┌─────────────────────┐         ┌─────────────────────┐
│ ✅ 通用编码规范       │         │ ✅ Python 类型提示     │
│ ✅ 安全规范           │         │ ☐ API 设计约定        │
│ ✅ Git 提交规范       │         │ ✅ 项目架构说明        │
│ ☐ React 最佳实践     │         │                      │
└─────────────────────┘         └─────────────────────┘
              ↘         ↙
         最终拼接预览 (只读)
    ┌──────────────────────────┐
    │ # 通用编码规范             │
    │ ...                       │
    │ # 安全规范                 │
    │ ...                       │
    │ # Python 类型提示          │
    │ ...                       │
    │ # 项目架构说明             │
    │ ...                       │
    └──────────────────────────┘
```

## 实施计划

### 8.1 数据结构
- electron-store 存全局拼图块列表
- 每个块: `{ id, title, content, tags }`
- 项目级通过 `.ruler/blocks.json` 存块启用状态 + 自定义块

### 8.2 UI 原型
- 左侧：项目列表下方新增"Global Rules"区
- 左侧：项目下新增"Project Rules"区
- 中间编辑器：选中的单个块编辑 / 或"全部拼接预览"
- 顶部标签切换：编辑模式 / 预览模式

### 8.3 实现
- RulePiece 数据结构
- BlockEditor 组件（块列表 + 块编辑）
- 拼接预览 Markdown
- Store 更新
